# 이메일 뉴스레터로 Macy's·The Outnet 딜 받기

## 왜 이 방법인가

Macy's(실측 84건)와 The Outnet(23건)은 Akamai 계열 봇 차단이 걸려 있어
직접 크롤이 불가능하다. 집 인터넷에서 재점검해도 뚫리지 않았다.

그런데 **두 사이트 모두 세일이 시작되면 구독자에게 메일을 보낸다.**
사용자가 직접 구독하고 본인 메일함을 본인 자격증명으로 읽는 것이므로
**리테일러 약관과 무관하다** — 사이트에 자동 접근하는 게 아니라
내가 받은 내 메일을 읽는 것이기 때문이다.

오히려 이 방식이 더 빠를 수 있다. 플래시 세일은 리테일러가 메일로 먼저
알리므로, 세일 페이지를 주기적으로 긁는 것보다 앞설 수 있다.

## 설정 (10분)

### 1. 전용 Gmail 계정 만들기

기존 메일과 섞이지 않게 새 계정을 하나 만드는 걸 권한다.
쇼핑 메일은 양이 많아서 개인 메일함에 섞으면 곤란하다.

### 2. 뉴스레터 구독

새로 만든 주소로 아래 사이트에서 이메일 구독을 신청한다.
보통 사이트 하단(footer)에 "Sign up for emails" 입력칸이 있다.

- **macys.com** — 우선순위 1 (실측 84건)
- **theoutnet.com** — 우선순위 2 (실측 23건)
- 여유가 되면: adidas.com, rei.com, levi.com, asos.com,
  saksfifthavenue.com, bloomingdales.com, net-a-porter.com, mrporter.com

구독 후 첫 메일이 오기까지 하루 정도 걸릴 수 있다.

### 3. Gmail 앱 비밀번호 발급

Gmail은 보안상 일반 비밀번호로 외부 프로그램 접속을 막는다.
'앱 비밀번호'라는 별도 16자리 비밀번호가 필요하다.

1. https://myaccount.google.com/security 접속
2. **2단계 인증**을 먼저 켠다 (이게 켜져 있어야 앱 비밀번호 메뉴가 나온다)
3. 검색창에 "앱 비밀번호" 입력 → 해당 메뉴 진입
4. 앱 이름 아무거나 입력 (예: `hotdeal-radar`) → 생성
5. **16자리 비밀번호**가 나온다 (공백 포함해 보이지만 붙여 써도 된다)

### 4. 환경변수 등록

명령 프롬프트를 열고 아래를 입력한다. 따옴표까지 그대로.

```
setx RADAR_IMAP_USER "새로만든주소@gmail.com"
setx RADAR_IMAP_PASS "앱비밀번호16자리"
```

입력 후 **열려 있는 명령창을 모두 닫는다.** 환경변수는 새 창부터 적용된다.

### 5. 확인

```
9_이메일수집.bat
```

구독한 메일에서 딜을 뽑아 점수와 함께 보여준다.
아직 메일이 안 왔으면 "총 0건"이 나오는데, 정상이다. 하루 뒤 다시 실행해 보면 된다.

## 어떻게 동작하나

수신 메일 중 **등록된 발신 도메인**(macys.com, theoutnet.com 등)에서 온 것만 검사한다.
개인 메일이 섞여 있어도 무시하므로 안전하다.

제목과 본문에서 할인율·쿠폰코드·링크를 추출한다. 할인 정보가 없는 메일
(신상품 안내, 배송 알림 등)은 딜로 보지 않고 건너뛴다.

### 할인율 계산이 조금 특별하다

쇼핑 메일은 보통 이런 형태다.

> **Extra 30% Off** Sale & Clearance — **Up to 70% off** select styles

여기서 30%만 보면 실제 할인폭을 크게 과소평가한다.
이미 70% 내려간 상품에 추가 30%가 붙으면 실효 할인은 **79%**다.

```
실효할인 = 1 - (1 - 0.70) x (1 - 0.30) = 0.79
```

이 계산을 반영하되, "up to"는 마케팅 문구라 전 품목에 적용되지 않으므로
**상한 85%**를 둬서 과대평가를 막는다.

이렇게 잡힌 딜은 제목에 `[세일공지]` 가 붙는다.
개별 상품가가 아니라 "이 사이트에 지금 큰 세일이 있다"는 신호라는 뜻이다.

  크롤이 되는 123개 사이트는 여전히 개별 상품 단위로 정확히 잡는다.
