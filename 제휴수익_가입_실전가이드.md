# 제휴수익 가입 실전 따라하기 가이드

> 목표: 딜온도가 뿌리는 딜 링크에 제휴 추적을 붙여, 클릭·구매마다 커미션을 받는다.
> 코드는 이미 다 만들어져 있습니다(`affiliate.py`). **이 문서는 "가입해서 ID 받아 넣는 것"만** 다룹니다.

---

## 0. 전체 그림 — 무엇을, 어떤 순서로

| 순서 | 서비스 | 성격 | 승인 속도 | 딜온도 연결 |
|---|---|---|---|---|
| ① 먼저 | **Skimlinks** | 스마트링크(한 번에 4.8만 머천트 자동 제휴) | 영업일 2일 | 시크릿 `RADAR_SKIMLINKS_ID` |
| ② 대안/추가 | **Sovrn Commerce** | 스마트링크(대안) | 최대 24시간~ | 시크릿 `RADAR_SOVRN_KEY` |
| ③ 나중 | **CJ Affiliate** | 고단가 머천트 직접 제휴(Mytheresa 등) | 머천트별 심사 | `data/affiliate.csv` 한 줄 |

**추천 순서**: ①로 전체를 한 번에 켜고(승인 쉬움·즉시 전 사이트 커버), 매출 큰 명품/백화점만 ③으로 직접 승인받아 커미션을 극대화. ②는 ①이 거절되거나 커버리지를 넓힐 때.

> 왜 Skimlinks가 먼저인가: Sovrn은 "이미 제휴링크가 깔려 클릭이 나는 사이트"를 요구해서 초기엔 닭-달걀 문제가 있습니다. Skimlinks는 **사이트 콘텐츠만 보고 심사**해서, 지금 있는 GitHub Pages 딜보드로 바로 신청 가능합니다.

---

## 1. 준비물 (공통)

1. **살아있는 딜보드(웹사이트)** — `https://bized...github.io/Dealondo/` 처럼 접속 가능해야 함. (상시가동 셋업으로 이미 확보. 심사에 이게 필요합니다.)
2. **이메일** (구글 계정)
3. **PayPal 계정** — 한국 거주자는 커미션을 PayPal로 받습니다(아래 5번).
4. **도메인/채널 소유 증명 수단** — 딜보드가 본인 것임을 보일 수 있어야 함(가입 이메일과 사이트 연결).

---

## 2. ① Skimlinks 가입 (핵심·먼저)

**a. 신청**
1. `skimlinks.com` → **Publishers → Sign up**.
2. 폼에 입력: 이름, 이메일, **사이트 URL = 딜보드 주소**, 사이트 성격(쇼핑/딜 큐레이션).
3. 제출.

**b. 심사 (영업일 2일)**
- Skimlinks가 사이트를 봅니다. 통과 기준: **접속 가능한 라이브 사이트 + 상품·브랜드를 언급하는 "가치 있는" 콘텐츠 + 소유 증명**.
- 이메일로 승인/추가요청이 옵니다.

**c. 승인 후 — 퍼블리셔 ID 확인**
1. **Publisher Hub** 로그인.
2. 계정/설정에서 **Publisher ID**(숫자) 확인. (사이트별 site ID도 있지만, 우리는 계정 Publisher ID를 씁니다.)

**d. 딜온도에 연결**
- GitHub 리포 → **Settings → Secrets and variables → Actions → New secret**:
  - 이름 `RADAR_SKIMLINKS_ID`, 값 = 방금 확인한 ID.
- 끝. **다음 자동 실행부터 전 딜 링크가 커미션 링크로 나갑니다.** (코드 수정 0)

> 거절되면? 흔한 사유는 "콘텐츠가 단순 집계뿐"입니다. 딜보드에 **한글 큐레이션 요소**(왜 좋은 딜인지, 브랜드 등급 ⭐/🔷, 관부가세 포함 실구매가 등 — 이미 있음)를 전면에 두면 "가치 있는 콘텐츠"로 인정받기 쉽습니다.

---

## 3. ② Sovrn Commerce 가입 (대안/추가)

1. `sovrn.com` → **Commerce → Sign up** 으로 계정 생성.
2. 두 경로:
   - **소셜 커넥터**(인스타·틱톡·페북 연결) → 자동 머천트 승인, **24시간 내** 활성.
   - **전통 심사**(웹사이트) → 더 엄격하지만 승인 시 수천 머천트 즉시 접근.
3. 주의: Sovrn은 **제휴 링크가 실제로 깔려 클릭이 발생하는 상태**를 심사 전에 요구합니다 → Skimlinks를 먼저 켜서 클릭을 만든 뒤 신청하면 유리.
4. 승인 후 **키/ID**를 받아 GitHub 시크릿 `RADAR_SOVRN_KEY`에 넣기.
   (Skimlinks·Sovrn 둘 다 넣으면 코드가 Skimlinks를 우선 사용합니다.)

- 지급: **PayPal $25부터**, 매월 말 기준 **90일 후** 정산.

---

## 4. ③ CJ Affiliate — 고단가 머천트 직접 제휴 (Mytheresa·노드스트롬 등)

스마트링크보다 커미션이 높습니다(중간 수수료 없음). 매출 큰 곳만 골라 하세요.

**a. 가입**
1. `cj.com` → **Publishers → Sign up**. (만 18세+, 본인 사이트 필요, 무료)
2. 이메일/비번 → 확인메일 → 프로필 작성(사이트 URL은 `https://www.` 형식 완전한 주소).

**b. 머천트 신청**
1. 로그인 → **Advertisers** 검색 → **Mytheresa** 신청. (노드스트롬·기타도 동일)
2. 각 광고주의 제휴매니저가 심사 → 수 시간~수일 내 승인/거절.

**c. 딥링크 만들어 딜온도에 등록**
1. 승인된 머천트에서 **Deep Link / Link Generator**로 상품URL→추적링크 형식을 확인. (내 Publisher ID가 박힌 `anrdoezrs.net/links/<PID>/type/dlg/...` 형태)
2. `data/affiliate.csv`에 한 줄 추가:
   ```
   mytheresa.com,CJ,https://www.anrdoezrs.net/links/<내PID>/type/dlg/sid/dealondo/{url}
   ```
   - `{url}` 자리에 상품 URL이 자동으로 인코딩되어 들어갑니다.
3. 커밋·푸시하면 그 머천트 딜은 **스마트링크 대신 직접 딥링크**로 나가 커미션 최대.

> 주의: CJ는 6개월간 실적이 전혀 없으면 계정을 비활성화할 수 있습니다. 딜보드가 돌아가면 문제없습니다.

## 5. 한국 거주자 — 세금과 수취 (중요)

- **세금 양식 W-8BEN**: 미국 기반 네트워크(CJ·Skimlinks/Sovrn)는 비미국 거주자에게 **W-8BEN**을 요구합니다. "나는 미국 납세자가 아니다"를 증명해 원천징수를 면제받는 양식입니다. 가입 중 온라인으로 작성(이름·한국 주소·거주국 KR 입력). PayPal 수취 시엔 PayPal이 대신 **외국인증명(CFS)**을 받기도 합니다. 유효기간 약 3년.
- **수취 수단**: 한국은 **PayPal**로 받습니다(Skimlinks·Sovrn 공통). 은행 직접입금은 US/EU/UK/JP 계좌만. PayPal 계정을 미리 만들어 두세요.
- **환전**: PayPal 잔액을 원화로 인출(연결 계좌/카드).

## 6. 승인 잘 받는 팁 (거절 방지)

1. **사이트가 반드시 라이브**여야 함(비번 걸린 페이지·준비중 페이지 불가). → 상시가동 켜두고 신청.
2. **집계만 있으면 거절 위험** → 딜보드에 원본성 요소를 보이게: 한글 딜 설명, 브랜드 등급(⭐/🔷), 관부가세 포함가, "왜 핫딜인지" 한 줄. (대부분 이미 구현돼 있으니 상단에 노출만)
3. **소유 증명** → 가입 이메일과 사이트가 연결돼 보이게(About/문의 이메일 표기 권장).
4. **연락 가능** → 승인팀이 추가정보를 요청할 수 있으니 메일 확인.

## 7. 승인 후 → 딜온도 연결 요약

| 서비스 | 넣는 곳 | 값 |
|---|---|---|
| Skimlinks | GitHub 시크릿 `RADAR_SKIMLINKS_ID` | 퍼블리셔 ID |
| Sovrn | GitHub 시크릿 `RADAR_SOVRN_KEY` | 커머스 키 |
| CJ 머천트 | `data/affiliate.csv` 한 줄 | 딥링크 템플릿 |
| 서브ID(선택) | 시크릿 `RADAR_AFF_SUBID` | 유입분석용 라벨(기본 dealondo) |

넣고 커밋/시크릿 저장하면 **다음 실행부터 자동 적용**. 딜보드엔 이미 제휴 고지 문구가 들어가 있습니다.

## 8. 현실적 기대치 (정직하게)

- **전환율 1~3%** 수준. 초기 트래픽이 작으면 수익도 작습니다 — 구독자·방문자가 늘수록 붙습니다.
- **명품 1건이 큼**: Mytheresa 8% → ₩400만 가방 1개 = 약 33만원. 프리미엄 큐레이션 전략이 여기서 값을 합니다.
- **정산이 느림**: 네트워크마다 60~90일 후 지급. 첫 입금까지 시간이 걸립니다.
- **스마트링크는 수수료를 뗌**(약 25%): 커버리지·편의 대가. 큰 머천트는 CJ 직접으로 이전.
