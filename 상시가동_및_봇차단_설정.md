# 딜온도 — 상시가동(24시간) & 봇차단 사이트 공략 설정

이 문서 하나로 두 가지를 끝냅니다.

1. **상시가동** — 내 PC를 꺼도 GitHub 서버가 20분마다 자동으로 딜을 수집하고, 딜보드·텔레그램을 갱신합니다. (서버비 0원)
2. **봇차단 사이트(메이시스·아웃넷·컬럼비아 등)** — 제휴 상품피드 + 이메일 뉴스레터로 합법 우회합니다.

---

## 1부. GitHub Actions 상시가동

### 원리
GitHub에는 **Actions**라는 자동 실행 기능이 있습니다. 공개 리포지토리는 이 기능이 **무료·무제한**입니다. 우리가 넣어둔 스케줄(20분마다)에 맞춰 GitHub 서버가 스크래퍼를 돌리고, 결과 `deals.json`을 리포에 커밋합니다. → GitHub Pages 딜보드가 자동 갱신되고, 텔레그램 알림도 서버에서 발송됩니다. **내 컴퓨터는 꺼져 있어도 됩니다.**

### 준비물
- 이미 쓰고 있는 GitHub 계정과 `Dealondo` 리포(공개 상태여야 함 — Actions 무료 조건)

### 단계

**① 프로젝트 전체를 Dealondo 리포에 올리기**

지금 리포에는 `index.html`·`deals.json`만 있습니다. 스크래퍼가 서버에서 돌려면 **프로젝트 전체**가 리포에 있어야 합니다. `hotdeal_radar` 폴더 안의 모든 것을 리포에 올리세요:

```
scraper/        (파이썬 코드)
data/           (brands.csv, feeds.csv, watchlist.csv)
web/            (딜보드 UI)
.github/workflows/radar.yml   (자동 실행 설정 — 이미 만들어 둠)
requirements.txt
.gitignore
index.html      (기존 것 — 자동으로 최신 UI로 갱신됨)
deals.json      (기존 것 — 자동 갱신됨)
```

GitHub Desktop을 쓰면: `Dealondo` 리포 폴더에 위 파일들을 복사 → Commit → Push.
(웹에서 업로드해도 되지만 폴더 구조가 유지되도록 GitHub Desktop을 권장합니다.)

**② 시크릿(비밀 키) 등록**

토큰을 코드에 넣으면 노출되므로 GitHub 금고에 넣습니다.
리포 → **Settings → Secrets and variables → Actions → New repository secret** 에서 아래를 하나씩 추가:

| 이름 | 값 |
|---|---|
| `TELEGRAM_BOT_TOKEN` | 봇 토큰 (BotFather에서 받은 것) |
| `TELEGRAM_CHAT_ID` | 내 chat id |
| `NAVER_CLIENT_ID` | 네이버 개발자센터 값 |
| `NAVER_CLIENT_SECRET` | 네이버 개발자센터 값 |

(이메일 수집을 쓸 거면 3부에서 `RADAR_IMAP_USER`, `RADAR_IMAP_PASS`도 추가)

**③ Actions 켜기**

리포 → **Actions** 탭 → 처음이면 "I understand my workflows, enable them" 클릭.
`Dealondo Radar` 워크플로가 보입니다. **Run workflow** 버튼으로 지금 한 번 수동 실행해 보세요.

**④ 확인**
- Actions 탭에서 실행이 초록불(성공)로 끝나는지 확인 (첫 실행은 5~8분: Playwright 설치 때문).
- 텔레그램에 알림이 오는지 확인.
- 딜보드(`https://bigstore222-hub.github.io/Dealondo/`)가 갱신됐는지 확인.

이후로는 **20분마다 자동**입니다. PC를 꺼도 됩니다.

> 참고: GitHub 스케줄은 서버 부하에 따라 몇 분~십몇 분 지연되거나 가끔 건너뛸 수 있습니다(무료의 특성). 촘촘한 실시간이 꼭 필요하면 2부의 유료 VM을 고려하세요.

## 2부. 봇차단 사이트 — 제휴 상품피드

메이시스·아웃넷·컬럼비아는 크롤링이 막혀 있습니다(403). 이들은 **제휴 네트워크에 상품피드**(전 상품 + 정가 + 세일가 + 링크 + 이미지)를 올려두며, 승인된 제휴사는 이걸 합법적으로 내려받습니다. 봇 우회가 아니라 **정식 제공 데이터**이고, 클릭 시 커미션까지 생깁니다.

### 어느 네트워크에 있나 (2025 기준 일반적 소속)
- **메이시스(Macy's)** → Rakuten Advertising
- **The Outnet / NET-A-PORTER** → Rakuten / Impact
- **컬럼비아(Columbia)** → CJ Affiliate (Commission Junction)
- 노드스트롬·기타 다수 → Rakuten / CJ / Impact 중 하나

### 절차
1. **제휴 네트워크 가입** (Rakuten Advertising 우선 권장 — 별도 문서 `Rakuten가입절차.md` 참고).
2. 네트워크 안에서 **원하는 광고주(메이시스 등)에 제휴 신청** → 승인 대기(보통 며칠).
3. 승인되면 광고주별 **Product Feed / Data Feed** 메뉴에서 **피드 URL**을 받습니다(CSV/TSV/XML).
4. 그 URL을 `data/feeds.csv`에 한 줄 추가:

```
name,retailer,url,format,category
Macys-Rakuten,macys.com,<받은_피드_URL>,auto,premium_fashion
TheOutnet-Impact,theoutnet.com,<받은_피드_URL>,xml,premium_fashion
Columbia-CJ,columbia.com,<받은_피드_URL>,csv,sports_outdoor
```

- `category`는 `premium_fashion / sports_outdoor / electronics / kids / etc` 중 하나(비우면 자동추론).
- 저장 후 커밋·푸시하면 **다음 실행부터** 자동으로 그 사이트 세일 상품이 딜보드에 들어옵니다.

시스템이 알아서: 세일가가 정가보다 낮은 상품만 추림 → 브랜드게이트·관부가세·점수 계산 → 발행. **코드는 손댈 필요 없습니다. feeds.csv 한 줄이면 끝.**

> 승인 전에는 feeds.csv가 비어 있어도 아무 문제 없습니다(그 소스만 0건).

## 3부. 봇차단 사이트 — 이메일 뉴스레터(보조)

제휴피드가 '전 상품'이라면, 이메일은 '지금 이 세일·이 코드'를 빠르게 잡는 보조 경로입니다.

1. 감시할 리테일러(메이시스·아웃넷 등)의 **뉴스레터를 구독**(전용 Gmail 계정 권장).
2. Gmail **앱 비밀번호**(16자리) 발급: Google 계정 → 보안 → 2단계 인증 → 앱 비밀번호.
3. GitHub 시크릿에 추가:
   - `RADAR_IMAP_USER` = 그 Gmail 주소
   - `RADAR_IMAP_PASS` = 16자리 앱 비밀번호
4. 끝. Actions가 T4 주기(6시간)마다 메일함을 읽어 **공개 세일·결제창 코드**를 뽑습니다.

> 신규가입/개인 전용 코드는 자동으로 걸러져 발행되지 않습니다(이미 구현됨). 공개 코드만 "🎟 결제창 코드"로 표시됩니다.

## 요약
- **상시가동**: 프로젝트를 리포에 올리고 → 시크릿 등록 → Actions 켜기. 그 뒤 PC 꺼도 20분마다 자동.
- **봇차단 사이트**: 제휴 승인받아 `data/feeds.csv`에 피드 URL 한 줄(주력) + 이메일 시크릿(보조).
- 둘 다 코드 수정 없이 **설정 파일·시크릿만으로** 붙습니다.
